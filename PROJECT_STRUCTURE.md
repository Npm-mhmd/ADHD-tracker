## Technology Stack

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT for authentication
- bcrypt for password hashing
- crypto-js for data encryption

### Frontend
- React.js (Vite)
- Tailwind CSS
- Recharts for data visualization
- Axios for API requests
